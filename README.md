### Animal (базовый класс) implements IAlive

── Herbo (травоядные)

── Predator (хищники)

### Thing (базовый класс для вещей) implements IInventory


## Ключевые архитектурные решения

_**Разделение ответственности:**_
* **IAlive** отвечает только за характеристики питания
* **Animal** содержит общие свойства всех животных
* Специализированные классы добавляют уникальное поведение

## Гибкость расширения:

* Легко добавить новые типы животных через наследование

* Просто расширить функциональность базовых классов

## Безопасность типов:

* Использование виртуальных методов обеспечивает правильное поведение
* Инкапсуляция данных защищает внутреннее состояние объектов

## Управление ресурсами:

* std::shared_ptr предотвращает утечки памяти
* RAII принцип для автоматического управления ресурсами
